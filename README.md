# BOMA

### Requirements:
- sass + compass
- node
- bower

### Local setup:
1. Clone repository
2. Install all dependencies:

```
$ npm install
$ bower install
```

### Gulp Tasks:

1. `gulp` or `gulp dev` - run development server
2. `gulp build` - build production package (preview)
3. `gulp deploy` - build & deploy to gh-pages branch
